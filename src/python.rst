Python

Installing
----------


Ubuntu 16.04
~~~~~~~~~~~~

::
    
    sudo add-apt-repository ppa:jonathonf/python-3.6
    sudo apt-get update
    sudo apt-get install python3.6

or::

    sudo add-apt-repository ppa:fkrull/deadsnakes
    sudo apt-get update
    sudo apt-get install python3.6

Ubuntu 16.10+
~~~~~~~~~~~~~

::

    sudo apt-get update
    sudo apt-get install python3.6
